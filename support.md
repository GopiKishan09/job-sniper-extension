# 🆘 Job Sniper Support

Welcome to the support page for **Job Sniper Chrome Extension**.

---

## 🚀 Getting Started

1. Open LinkedIn Jobs page
2. Click on the Job Sniper extension
3. Enable auto-refresh
4. The page will refresh automatically to show latest jobs

---

## ❓ Common Issues

### 🔄 Auto-refresh not working

* Make sure you are on the LinkedIn jobs page
* Check if the extension is enabled
* Try reloading the extension

---

### ⚙️ Extension not loading

* Go to `chrome://extensions`
* Enable Developer Mode
* Reload the extension

---

### 🔔 Notifications not showing

* Check Chrome notification permissions
* Ensure notifications are enabled in your browser
