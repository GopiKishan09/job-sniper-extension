# Privacy Policy

**Job Sniper Chrome Extension**

Last updated: April 2026

---

## 🔍 Overview

Job Sniper is a Chrome extension designed to enhance the LinkedIn job search experience by providing features like auto-refresh and improved browsing.

We respect your privacy and are committed to protecting your data.

---

## 📊 Data Collection

This extension **does NOT collect, store, or share any personal data**.

We do not:

* Track user activity
* Collect personal information
* Send data to external servers

---

## ⚙️ Permissions Usage

The extension requires the following permissions:

* **activeTab**
  Used to interact with the currently active LinkedIn job page.

* **tabs**
  Used to refresh the active tab when auto-refresh is enabled.

* **storage**
  Used to save user preferences (e.g., auto-refresh on/off).

* **alarms**
  Used to schedule automatic refresh intervals.

* **notifications**
  Used to notify users when the page is refreshed.

---

## 🌐 Host Permissions

* Access is limited to:
  `https://www.linkedin.com/*`

This is required to enhance the LinkedIn job browsing experience.

---

## 🔐 Data Security

Since no personal data is collected or transmitted, there is no risk of data leakage from this extension.

---

## ❗ Third-Party Services

This extension does not use any third-party analytics, tracking tools, or external APIs.
